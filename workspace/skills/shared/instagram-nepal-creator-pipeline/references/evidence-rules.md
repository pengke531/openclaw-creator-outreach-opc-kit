# Evidence Rules

## Geography pass

Pass only when Nepal is clear from one or more strong signals:

- profile explicitly says Nepal
- account location is Nepal
- repeated profile and content signals tightly anchor the creator to Nepal

Fail when:

- another country is explicit

Use `EVIDENCE_GAP` when:

- Nepal cannot be established clearly enough

## Personal creator pass

Pass only when the account is clearly person-led:

- creator
- influencer
- blogger
- public figure
- personal blogger
- strong personal-name identity with personal content and self-description

Fail when:

- media brand
- organization
- repost page
- community page
- aggregator
- no clear personal identity

## Followers pass

Pass only when the visible Instagram profile follower display is at least
`100000`.

Record the visible display value directly, even if rounded.

## Registration rule

No pass on partial evidence.

If geography, identity, or followers are unclear:

- do not register
- do not submit
- save evidence packet
- mark result as `EVIDENCE_GAP`
