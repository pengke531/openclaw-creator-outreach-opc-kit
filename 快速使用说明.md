# 快速使用说明

前提：对方已经安装并能正常使用 OpenClaw。

## 当前推荐方式

不要再依赖导入脚本。

正确交付方式是：

1. 把这个仓库拉到客户本地
2. 在仓库目录打开 Codex
3. 让客户本地 Codex 按 `MANUAL_CONFIGURATION_SPEC.md` 手动整理和配置本地架构

## 当前目标架构

- `main` = Manager
- `wangcai` = 侦察与证据执行
- `laicai` = 外联执行
- `facai` = ROI 复盘执行

## 系统层

- `workspace/registry`
- `workspace/evidence`
- `workspace/inbox`
