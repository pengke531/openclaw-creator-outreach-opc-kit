# 快速使用说明

前提：对方已经安装并能正常使用 OpenClaw。

## 给别人时，直接这样说

### Windows

```powershell
git clone https://github.com/pengke531/openclaw-creator-outreach-opc-kit
cd openclaw-creator-outreach-opc-kit
powershell -ExecutionPolicy Bypass -File .\import.ps1
```

### macOS / Linux / WSL2

```bash
git clone https://github.com/pengke531/openclaw-creator-outreach-opc-kit
cd openclaw-creator-outreach-opc-kit
chmod +x ./import.sh
./import.sh
```

## 只要记住这一句话

已经有 OpenClaw 的用户，只需要拉仓库，然后运行 `import`。

## import 会自动做什么

- 预检
- 导入架构
- 合并配置
- 创建领域目录
- 注入 4 个 Agent
- 校验配置
- 如果 gateway 正在运行，自动完成基础验证

## 如果执行时提示 gateway 没启动

先运行：

```bash
openclaw gateway
```

然后重新执行同一个 `import` 命令。
